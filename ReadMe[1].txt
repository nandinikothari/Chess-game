Description

This is a chess game implemented as a web application using HTML, CSS, and JavaScript. You can play chess against a friend in the browser with this easy-to-use web chess game.


Features
- Two-player mode for playing with a friend.
- Intuitive user interface with drag-and-drop functionality.
- Highlights legal moves and en passant capture.
- Implements standard chess rules, including castling and promotion.
- Responsive design for different screen sizes.

Getting Started

Prerequisites
- A modern web browser (Chrome, Firefox, Safari, etc.).

Installation
1. Clone the repository or download the source code as a ZIP file.
2. Extract the ZIP file (if downloaded).
3. Open `index.html` in your web browser.

How to Play
1. Click on a piece you want to move.
2. Drag it to the square where you want to place it.
3. Release to make the move.
4. Follow the standard rules of chess.

Game Rules
This chess game follows the standard rules of chess. You can refer to various online resources or books to understand these rules. Here are some key points:

- Pawns move forward one square but capture diagonally.
- Knights move in an L-shape.
- Bishops move diagonally.
- Rooks move horizontally or vertically.
- Queens can move horizontally, vertically, or diagonally.
- Kings can move one square in any direction.
